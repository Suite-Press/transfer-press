Transfer Press is the ultimate plugin management solution for WordPress. Easily export, import, download, move, and clone plugins between sites without FTP or manual file handling.

== Description ==

**Transfer Press** is your one-stop WordPress plugin for seamlessly exporting, importing, downloading, and transferring plugins between WordPress installations. Whether you're a developer moving plugins from staging to production, an agency deploying consistent plugin stacks, or a site owner creating backups, Transfer Press streamlines your workflow.

Gone are the days of dealing with FTP, zip tools, or file managers. With Transfer Press, you can:

- Export any installed plugin as a clean ZIP archive with one click.
- Import plugin ZIP files and optionally activate them instantly.
- Clone plugin setups between WordPress sites effortlessly.
- Speed up staging-to-live migrations and restore processes.

**Ideal for:**
- WordPress developers
- Agencies managing multiple client sites
- Freelancers performing regular migrations
- Non-technical users needing simple plugin backups or transfers

== Installation ==

1. Upload the plugin to `/wp-content/plugins/transfer-press` or install it via the WordPress Plugin Directory.
2. Activate it from the **Plugins** menu in WordPress.
3. Navigate to **Transfer Press > Export** to begin exporting plugins.
4. Use **Transfer Press > Import** to upload and install plugin ZIP files.

== Usage ==

1. Go to **Transfer Press** from your WordPress dashboard.
2. To export: Choose a plugin, click **Export**, and download the generated ZIP file.
3. To import: Upload a plugin ZIP file and click **Import**. The plugin will be installed and optionally activated.

== Frequently Asked Questions ==

= Can I use this to migrate plugins between two WordPress sites? =
Absolutely! Transfer Press was built specifically for migrating plugins from one WordPress site to another.

= Will the plugin be activated automatically after importing? =
Yes, you have the option to activate it automatically after import.

= Can I use this on multisite WordPress networks? =
Yes. Transfer Press supports both single-site and multisite environments.

= Are premium plugins supported? =
Yes, you can transfer premium plugins as ZIPs, but you must comply with their license terms.

= Is FTP required to use this plugin? =
No FTP is needed. Transfer Press handles everything through the WordPress dashboard.
